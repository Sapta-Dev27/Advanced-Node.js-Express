# Key Features of Nodejs 
1. Open Source Tool 
2. Cross Platform Tool ( works on all devices : MAC , WINDOWS)
3. Uses chrome's v8 engiene to run JS code outside the browser 
Hence , Nodejs is basically neither a FRAMEWORK , nor a LIBRARY  , nor a LANGUAGE . It is basically a JS runtime environment made using 
chrome's v8 engiene to run JS code outside browser . It make JS code to run on the server side .
4.Nodejs is basically not suitable for those task where it requires high CPU intensive .
5. Nodejs follows the event driven architechture : single threaded , non  block IO model 

# Difference of Nodejs and JS 
1. Nodejs => runs on server , database connectivity possible and file system access is possible 
2. Javascript => runs on browser , database connectivity is not possible and file system access is also restricted
